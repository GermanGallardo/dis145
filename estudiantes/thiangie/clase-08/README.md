# clase-08
## Breve encargo
### 1. Lista preliminar de los materiales que se tiene (en relación al proyecto)
1. LEDs
2. 3-4 botones
3. Cables tipo M-M
4. Cables tipo M-H
5. Potenciómetro

### 2. Lista de los materiales que se deben conseguir (en relación al proyecto)
1. Más botones para alcanzar los 7 (extensión esperada del proyecto)
    · Dependiendo del presupuesto disponible:
      · Sensor de flexión
      · Sensor de pulso cardiaco (disponible en FabLab)
      · Sensor de sonido
      · Sensor de golpes
3. Impresiones 3D / Cortes MDF (motivos estéticos)
4. Cables tipo H-H para poder extender los cables
5. Paquete cables tipo M-M y H-M
6. LEDs RGB

### 3. El proyecto explicado en una frase
Sintetizador de emociones a través del color y la música.

### 4. Extensión de la explicación del proyecto
A través de la interacción con distintos sensores y botones en Arduino se espera generar una interfaz que interprete las acciones realizadas como sonidos que también se representan de manera simultánea. Esto con la finalidad de poder "sintetizar" emociones.
